# Feedback Linearization Controller

Implements Feedback Linearization Controller for position trajectory tracking of Quadcopter X500 on Gazebo Harmonic-PX4-ROS2 Jazzy environment.

### Environment Setting
- ROS2 Jazzy
- Gazebo Harmonic
- PX4 Autopilot
